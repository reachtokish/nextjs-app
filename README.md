# NextJs-app
App built on Next.js, a React Framework

## Technologies Used
- React.Js
- Next.Js

## To run the App
```sh
git clone https://github.com/reachtokish/nextjs-app.git
cd nextjs-app
npm install
npm run dev
```
App should run on http://localhost:3000/

## To build the App
```sh
npm run build
```
